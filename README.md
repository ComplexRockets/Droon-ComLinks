# Droon-ComLinks
by Complex Rockets
Version: Beta [yet to be released]

This SimpleRockets 2 mod intends to add a new level of difficulty and realism to the game by adding a communication mechanism. The player will need to construct a network of relay satellites to explore the Juno System beyond a Low Droo Orbit. Loss of signal will disable remote control and the craft will have to be controlled by on-board droon or by a vizzy program. For an added difficulty the player can also activate a realistic control delay when far from Droo.

Any contributor is welcome, join the Complex Rockets Community at discord https://discord.gg/cdJmWeb if you are interested in modding SR2
